## Source: William Lidwell, Kritina Holden, Jill Butler — 3rd Edition (200 Principles)

## Key Principle for UX Auditing

This reference covers 200 design principles spanning usability, cognition, perception, aesthetics,
and interaction design. Each principle below includes its core definition, UX implication, and application.

✅ Use these principles to identify root causes in UI/UX issues.
✅ Cite principles by name when identifying problems in audits.
✅ Prioritise perception, cognition, and interaction principles for screen-based UI audits.

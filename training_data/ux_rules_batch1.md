# UX Rules — Batch 1 (Parts 01-05)

## RULE: Keep dividers inside the grid
**Chapter:** Accordion
**✅ Correct:** Dividers are contained within the accordion grid, keeping visual boundaries aligned with content.
**❌ Wrong:** Dividers extend outside the grid, breaking visual alignment.
**Key fix:** Contain accordion dividers within the grid boundaries.
---

## RULE: Communicate interactivity with the hover state
**Chapter:** Accordion
**✅ Correct:** The hovered accordion row shows a pointer cursor, making the interactive state clear.
**❌ Wrong:** No hover state change, leaving users uncertain if the row is clickable.
**Key fix:** Add hover state (cursor + visual feedback) to accordion rows.
---

## RULE: Provide accordion icons to indicate section's state
**Chapter:** Accordion
**✅ Correct:** Each accordion row has a + icon, clearly signaling it can be expanded.
**❌ Wrong:** No icons on accordion rows, so users can't tell if items are expandable.
**Key fix:** Add expand/collapse icons to every accordion row.
---

## RULE: Stick to familiar accordion icons
**Chapter:** Accordion
**✅ Correct:** Standard + icon is used for accordion rows, matching user expectations.
**❌ Wrong:** A star icon is used, which is unfamiliar and confusing as an expand indicator.
**Key fix:** Use universally recognized +/− icons for accordion expand/collapse.
---

## RULE: Add enough space between containers
**Chapter:** Accordion
**✅ Correct:** Adequate spacing between accordion items makes each row easy to distinguish and tap.
**❌ Wrong:** Rows are too close together, making them hard to differentiate and tap accurately.
**Key fix:** Increase vertical spacing between accordion containers.
---

## RULE: Keep sections simple in style
**Chapter:** Accordion
**✅ Correct:** Expanded accordion section shows only the question title and plain body text.
**❌ Wrong:** Expanded section contains colored links and heavy text styling, distracting from content.
**Key fix:** Keep expanded accordion content minimal — avoid decorative styling inside sections.

**Chapter:** Accordion
